# Offline-Travel-Guide

AIM

To develop an Android application using MIT App Inventor that provides travel-related information such as places, maps, and basic guides without requiring an internet connection, enabling users to access important travel details anytime and anywhere.

Description

The Offline Travel Guide is an Android application developed using MIT App Inventor.
It provides users with essential travel information such as popular places, descriptions, and images without requiring an internet connection.

This app is especially useful for travelers in areas with limited or no network access, allowing them to explore destinations anytime, anywhere.

ALGORITHM

Open MIT App Inventor.

Create a new project OfflineTravelGuide.

Design the screen:

Add ListView for places

Add Labels and Image for details

Add Buttons for navigation

Store travel data:

Save place names, descriptions, and images using TinyDB or assets

Write logic in Blocks:

Load places into ListView when app starts

When a place is selected, display its details

Ensure offline use:

Store all data inside the app (no internet required)

Test the app using AI Companion

Build APK and install on phone

End

📍 Offline Access – No internet required
